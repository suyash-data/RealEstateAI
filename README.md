# 🏠 AI-Powered Real Estate Market Intelligence Platform

An end-to-end Data Science project that predicts house prices using Machine Learning and provides interactive real estate analytics through a Streamlit dashboard.

## 🚀 Live Demo

https://realestateai-vdk6mffaqvdtsgtqb5nanw.streamlit.app/

## 📂 GitHub Repository

https://github.com/suyash-data/RealEstateAI

---

## 📌 Project Overview

This project analyzes real estate data and uses a Random Forest Regressor model to estimate house prices based on various property features.

The application includes:

* House Price Prediction
* Dataset Explorer
* Interactive Visualizations
* Feature Importance Analysis
* Streamlit Web Dashboard

## 📊 Dataset Information

Dataset Size: 545 Properties

Features:

* Area
* Bedrooms
* Bathrooms
* Stories
* Main Road Access
* Guest Room
* Basement
* Hot Water Heating
* Air Conditioning
* Parking
* Preferred Area
* Furnishing Status

Target Variable:

* House Price

## 📈 Dashboard Features

* Bedrooms vs Price Analysis

Shows which property features most influence house prices.
